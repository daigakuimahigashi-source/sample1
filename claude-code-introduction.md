# Claude Code（クロード・コード）ってなんやねん？ やさしく解説するで

## はじめに

「Claude Code」っちゅうんは、Anthropic社が作った **AIコーディングアシスタント** やねん。
ターミナル（コマンドライン）の上で動いて、あんたのプログラミング作業をめっちゃサポートしてくれるんよ。

ひとことで言うたら、**「AIが隣に座って一緒にコード書いてくれる」** みたいなツールやな。

---

## どんなことできるん？

### 1. コードを読んで理解してくれる

プロジェクトのファイルを読み込んで、コードの中身をちゃんと理解してくれるねん。
「この関数なにしてるん？」って聞いたら、わかりやすう教えてくれるで。

### 2. コードを書いてくれる

新しい機能の追加とかバグの修正を、普通の言葉でお願いするだけでやってくれんねん。

> **例：** 「ログイン機能つけてや」「このバグ直してくれへん？」

### 3. ファイルの編集・作成

今あるファイルを編集したり、新しいファイル作ったりもできるで。
変更内容はぜんぶ確認してから反映されるから、安心してや。

### 4. コマンドの実行

テスト走らせたり、ビルドしたり、Gitの操作したり、ターミナルでやる作業も代わりにやってくれるんよ。

### 5. Git操作のサポート

コミットメッセージ作ったり、プルリクエスト作ったり、Git関連の作業もスイスイこなすで。

## 今までのAIチャットとなにが違うん？

| 特徴 | 今までのAIチャット | Claude Code |
|------|-----------------|-------------|
| 動く場所 | ブラウザの上 | ターミナルの上 |
| ファイル操作 | でけへん | 直接読み書きできる |
| コマンド実行 | でけへん | 実行できる |
| プロジェクト理解 | コピペせなあかん | フォルダ丸ごと把握 |
| 変更の適用 | 手でコピペ | 自動で反映 |

一番でっかい違いは、**あんたのプロジェクトに直接アクセスできる** っちゅうとこやな。
コードをコピペしてチャットに貼り付ける、あの面倒な作業はもういらんで。

## 使い方のイメージ

```
# ターミナルでプロジェクトのフォルダに移動して起動するんや
cd my-project
claude

# あとは普通に話しかけるだけやで
> このプロジェクトの構成教えてや
> ユーザー登録のAPI作ってくれへん？
> テスト走らせて
> 変更コミットしといて
```

会話するみたいに、やりたいこと言うだけでOKやねん。めっちゃ楽やろ？

- **プログラミング初心者** → コードの書き方教えてもらいながら進められるで
- **経験者** → めんどい作業は任せて、大事な設計に集中できるわ

## 安全なん？

ほんま大事なとこやけど、ちゃんと安全に作られてるで。

- **確認してから実行：** ファイルの変更とかコマンド実行する前に、ちゃんと「ええか？」って聞いてくれる
- **ローカルで動く：** あんたのPC上で動くから、コードが勝手にどっかに送られたりせえへん
- **元に戻せる：** Gitと連携してるから、「あかん！」ってなってもいつでも戻せるで
