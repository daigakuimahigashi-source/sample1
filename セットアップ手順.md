# Google Drive PDF OCR 自動リネーム - セットアップ手順

## 1. Google Apps Script プロジェクトの作成

1. [Google Apps Script](https://script.google.com/) にアクセス
2. 「新しいプロジェクト」をクリック
3. プロジェクト名を「PDF自動リネーム」に変更

## 2. コードの貼り付け

1. デフォルトの `コード.gs` の内容を全て削除
2. `Code.gs` の内容をコピーして貼り付け
3. `Ctrl+S` で保存

## 3. appsscript.json の設定

1. 左側メニューの歯車アイコン「プロジェクトの設定」をクリック
2. 「"appsscript.json" マニフェスト ファイルをエディタで表示する」にチェック
3. エディタに戻り `appsscript.json` を開く
4. 内容を `appsscript.json` ファイルの内容で置き換え
5. 保存

## 4. Drive API の有効化

1. 左側メニューの「サービス」の横にある「＋」をクリック
2. 一覧から「Drive API」を選択
3. バージョン: `v2` を選択
4. 「追加」をクリック

## 5. 対象フォルダの準備

1. Google Drive で請求書PDFを入れるフォルダを作成
2. フォルダのURLからフォルダIDをコピー
   - URL例: `https://drive.google.com/drive/folders/XXXXX`
   - `XXXXX` の部分がフォルダID

## 6. 初回設定の実行

1. スクリプトエディタで関数選択を `setup` に変更
2. 「実行」ボタンをクリック
3. 権限の承認を求められたら許可する
4. ダイアログにフォルダIDを入力してOK

## 7. 実行

1. 対象フォルダにPDFファイルを配置
2. スクリプトエディタで関数選択を `main` に変更
3. 「実行」ボタンをクリック
4. 「実行ログ」で結果を確認

## リネーム形式

```
yyyymmdd_取引先名_¥000,000.-.pdf
```

例: `20260315_株式会社ABC_¥1,234,567.-.pdf`

## 抽出パターン

| フィールド | 対応するキーワード |
|-----------|-------------------|
| 請求期日 | 請求日、請求期日、お支払期限、振込期限、支払期日、支払期限 |
| 取引先名 | 「〇〇 御中」「〇〇 様」、請求先、取引先 |
| 金額 | 合計金額、ご請求金額、請求金額、お支払金額、合計、¥マーク |

## オプション: スプレッドシートから実行

1. 新しいGoogleスプレッドシートを作成
2. 「拡張機能」→「Apps Script」からスクリプトを開く
3. 同様にコードを貼り付け
4. スプレッドシートを再読み込みすると「PDF リネーム」メニューが表示される
